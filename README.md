## 环境依赖
1. Nodejs v18或更高
2. 启用 [Corepack](https://nodejs.cn/api/corepack.html) 或者安装 pnpm@8.15.4

## 安装依赖
> pnpm install

## 本地调试
> pnpm dev

## 打包部署
> 打包产物 `dist.zip`
```shell
pnpm build
```
